El código HTML del ejercicio del curso ahora incluye el atributo `alt` en las etiquetas ``, lo cual es una buena práctica en términos de accesibilidad y SEO. El atributo `alt` se utiliza para proporcionar una descripción de texto de la imagen, lo que es útil para las personas que utilizan lectores de pantalla y para los motores de búsqueda. En este caso, el atributo `alt` contiene "Jose Vicente Carratala," que describe la imagen. Este atributo ayuda a mejorar la accesibilidad de la página web al proporcionar una descripción adecuada de la imagen en caso de que no se cargue la imagen o para personas con discapacidades visuales. También puede ser útil para fines de indexación por motores de búsqueda, ya que les permite entender mejor el contenido de la imagen. Es una buena práctica incluir atributos `alt` descriptivos en todas las imágenes de tu sitio web.
